Pet API is a web api that allows users to store their pets as well as their own information. This API does not have security implemented so an API key is not required. Only simple data is transferred so loads over 200k will not be accepted by the API, as well as the data validation is not 100%.
https://petsapi4537.herokuapp.com/api/v1

Pets

Pets come in a variety of species

Pet Endpoints

Endpoint Description
GET

/pets/get

Gets all pets
GET

/pets/get/petid

Gets pets by id
PUT

/pets/update

Updates a pets information
DEL

/pets/delete

Deletes a pet
POST

/pets/create

Creates a new pet
